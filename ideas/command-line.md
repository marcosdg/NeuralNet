## Terminal interaction


* Help

```bash
neuralnet -help
```

* Loading configuration and data files

```bash
neuralnet -load net_dir net_file proben_dir proben_file [proben_files]
```

* Training the net

```bash
neuralnet -train number_of_runs
```

* Printing statistics

```bash
neuralnet -stats
```
