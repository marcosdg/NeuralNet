			COMMAND-LINE INTERACTION

0. Help:

```bash
neuralnet -help
```

1. Loading configuration and data files:

```bash
neuralnet -load net_dir net_file proben_dir proben_file [proben_files]
```
2. Training the net:

```bash
neuralnet -train
```
3. Printing statistics:

```bash
neuralnet -stats
```
