## Terminal interaction

1. Help

```bash
neuralnet -help
```

2. Loading configuration and data files

```bash
neuralnet -load net_dir net_file proben_dir proben_file [proben_files]
```

3. Training the net

```bash
neuralnet -train
```

4. Printing statistics

```bash
neuralnet -stats
```
